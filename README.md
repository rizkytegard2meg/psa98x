# FS-Blog 基于 SpringBoot 的个人博客系统
[![Hex.pm](https://img.shields.io/hexpm/l/plug.svg)](./LICENSE)

### 用到的技术
- SpringBoot
- MyBatis - ORM 框架
- MyBatis 通用 Mapper - 简化 SQL 操作 
- SpringMVC - MVC 最佳实践
- Freemarker - 模板引擎
- Lombok - 编译插件
- BootStrap 4.0 - 前端 CSS 框架
- Editor.md - 前端 Markdown 编辑器
- MySQL - 数据库
